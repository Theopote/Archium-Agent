status: Literal[
        "pass",
        "pass_with_warnings",
        "needs_review",
        "blocked",
    ]
```

建筑图纸额外检查：

* 图纸边界完整；
* 不发生 cover 裁剪；
* 指北针仍存在；
* 比例尺仍存在；
* 图例未缺失；
* 来源和图注未缺失；
* 图片身份未变化；
* 尺寸比例没有异常变化。

像素相似度不得作为唯一判断标准。

# 三、架构边界

本轮不得破坏以下边界。

## 1. Domain 层不得依赖 Streamlit

所有：

* ExportPolicy；
* SlideDesignBrief；
* ModelRole；
* SlideRecoveryResult；
* ExportRoundTripReport；

必须放在 Domain 或 Application 层。

---

## 2. UI 只负责交互，不直接编排复杂业务

禁止在 Streamlit 页面中直接：

* 调用多个 Provider；
* 操作文件；
* 修改数据库实体；
* 拼装导出 Manifest；
* 计算质量门。

必须通过 Application Service。

---

## 3. 所有长任务必须使用现有后台工作流

以下操作必须支持异步工作流状态：

* 逐页设计摘要生成；
* 外部页面恢复；
* Round-trip Validation；
* 批量导出；
* 图片背景清理；
* OCR；
* VLM 页面分析。

状态必须进入：

```text
QUEUED
RUNNING
AWAITING_REVIEW
SUCCEEDED
FAILED
CANCELLED
```

---

## 4. 所有外部模型调用必须可追踪

至少记录：

```text
role
provider
model
request_id
workflow_run_id
slide_id
duration
retry_count
success
failure_type
```

不得记录明文 API Key。
