# v0.2 Beta Backlog — 范围冻结

> **规则：非 blocker 不进本轮。** 本清单在 Beta 验证结束前冻结；新功能、重构、体验优化一律进入「Deferred」，除非能证明会阻断 Golden Case 或主路径 onboarding。

## 当前主产品（已冻结）

| 项 | 定义 |
|----|------|
| **产品形态** | Streamlit 项目工作台（`archium` / `streamlit run app.py`） |
| **核心链路** | 资料 → 事实 → Brief → Storyline → SlideSpec → 审核 → 导出 |
| **默认安装入口** | `archium`（`pyproject.toml` → `archium.cli:main`） |
| **Legacy** | `archium-legacy` / `main.py` — v0.1 实验 CLI，文档与命令均标注 legacy |

验收门槛：**从未接触过项目的人，按 README Quickstart 能走到同一条主路径。**

## Out of scope（本轮明确不做）

以下事项**即使有价值**，在 Beta 冻结期间也不排入本轮：

| 类别 | 示例 |
|------|------|
| 新 Stage / 新节点 | Stage 17、新 Agent、新导出格式 |
| Legacy 深化 | 文件整理 UX、指令中心持久化 |
| 模板 / 母版系统 | 建筑事务所级 PPT 母版、可替换占位符体系 |
| 视觉 QA  corpus 工具链 | 标注 corpus 生成 UI、校准面板（除非 Golden 阻断） |
| 依赖安全 enforcement | pip-audit 从观察期切为 merge blocker（见 [ci-security-audit.md](ci-security-audit.md)） |
| 生产部署脚手架 | Docker / K8s / 多租户 |
| API 公开化 | REST / gRPC 对外服务 |
| 视觉回归 baseline | ✅ L1 三 Case PNG 基线（`tests/golden/visual`） |
| `run_pipeline()` 移除 | ✅ v0.2-beta 准备阶段已移除；使用 `PresentationWorkflowService.run()` |
| Legacy 物理迁移 | 将 `file_manager.py` 等移入 `legacy/` 包（文档已标注，代码迁移延后） |
