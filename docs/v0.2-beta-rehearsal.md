# v0.2 Beta Rehearsal — 真实用户闭环

> **目标：** 把「至少一位非开发用户走通主链并记录修改成本」从口号变成**可汇总的数据**，并据此决定能否打 Beta 标签。

## 参与要求

| 项 | 要求 |
|----|------|
| 人数 | 1–3 名**非开发**用户（建筑师 / 规划师 / 项目经理） |
| 项目 | 脱敏真实建筑项目资料（PDF/DOCX/PPTX/图片） |
| 环境 | **Windows + Microsoft PowerPoint**（PPTX 终检权威环境） |
| 入口 | 仅 [`README` Quickstart](../README.md)：`archium` → 项目工作台 |
| 禁止 | 工程师代操作生成步骤；可旁听记录问题 |

## 正式流程（单次 session 90–120 分钟）

1. **准备（10 min）** —  facilitator 安装 `pip install -e ".[full]"`，配置 `.env`，准备脱敏资料包。
2. **导入（15 min）** — 用户自行创建项目、上传资料、确认片段/事实账本。
3. **生成（20 min）** — 用户填写 Brief → 审核 → Storyline → SlideSpec → 触发工作流导出（JSON + 原生 PPTX 或 Marp，按 rehearsal 脚本统一）。
4. **编辑（30–45 min）** — 用户在 Streamlit **与 PowerPoint** 中修改；记录每页类别与耗时。
5. **导出终检（10 min）** — 用户确认最终 PPTX/PDF 是否可汇报；PowerPoint 打开无阻断错误。
6. **复盘（15 min）** — 归类问题为三档（见下），填写 triage 表。

## 数据模板（必填）

新建 session 目录（复制模板）：

```bash
python scripts/new_beta_session.py 2026-07-18-session1
```

或手动复制到 `docs/rehearsal/sessions/<session_id>/`：

| 文件 | 用途 |
|------|------|
| [`templates/beta-edit-cost-sheet.csv`](templates/beta-edit-cost-sheet.csv) | 每页修改类别 + 分钟数 |
| [`templates/beta-issue-triage.csv`](templates/beta-issue-triage.csv) | 问题三档归类 |

汇总命令：

```bash
python scripts/summarize_beta_rehearsal.py docs/rehearsal/sessions/
```

## 修改类别定义

| 类别 | 包含 |
|------|------|
| `text` | 标题、正文、措辞、错别字 |
| `layout` | 溢出、对齐、间距、版式 |
| `image` | 素材匹配、裁切、替换、比例 |
| `fact_citation` | 数字、单位、引用、Fact Ledger |
| `structure` | 页序、章节、叙事逻辑 |
| `export` | 导出失败、格式、PowerPoint 兼容 |
| `other` | 上述未覆盖 |

## 问题三档

| 档位 | 定义 | 处理 |
|------|------|------|
| **beta_blocker** | 不修复则**不能**对外称 Beta | 进入 [v0.2-beta-backlog.md](v0.2-beta-backlog.md) In scope，阻断发版 |
| **post_beta_improvement** | 影响体验但不阻断 Beta | Deferred 或 v0.2.x 小版本 |
| **future_idea** | 新能力 / 长期优化 | 不进入 Beta 承诺 |

## Session 完成标准（对应 B10）

- [ ] 至少 1 名非开发用户独立完成：导入 → 生成 → 编辑 → 导出
- [ ] `beta-edit-cost-sheet.csv` 覆盖 **≥80%** 最终 deck 页面
- [ ] `beta-issue-triage.csv` 中所有 **critical/high** 已归类
- [ ] 汇总脚本输出 `summary.json` 并附入 [v0.2-beta-release-decision.md](v0.2-beta-release-decision.md)
