# 汇报工作室用户指南

**汇报工作室**（侧栏路径 `studio`）是 Archium v0.2 的**主编辑界面**：在同一页面完成页面浏览、版式预览、内容微调、视觉修改与 PowerPoint 导出。普通用户无需了解底层技术术语。

## 适用场景

| 你想做的事 | 推荐入口 |
|------------|----------|
| 浏览整套汇报、逐页检查版式 | **汇报工作室** |
| 减少文字、放大主图、切换版式类型 | **汇报工作室** → AI 编辑 |
| 缩短文字、拆页、转为要点 | **汇报工作室** → 内容适配 |
| 导出可编辑 `.pptx` | **汇报工作室** → 导出 PPTX |
| 审核视觉方向、在多个候选版式间挑选 | **视觉设计**（进阶） |
| 从零生成 Brief / Storyline / 页面内容 | **项目工作台** |

## 前置条件

1. **已有项目**，并在 **项目工作台** 完成内容管线：Brief → Storyline → **页面内容**（至少有一页 SlideSpec）。
2. （推荐）在 **设置** 或 `.env` 中配置 LLM API Key；未配置时部分生成步骤会降级为规则模式，Studio 仍可生成版式与导出。
3. **导出 PowerPoint** 需本机安装 Node.js，并在 `archium/infrastructure/renderers/pptxgen` 目录执行过 `npm install`（首次部署时一次性操作）。

## 界面概览

打开 **汇报工作室** 后，界面分为四个区域：

```
┌─────────────────────────────────────────────────────────────┐
│  项目 / 汇报选择 · 版式就绪指标 · 操作栏（生成 / 保存 / 导出）│
├──────────┬────────────────────────────┬─────────────────────┤
│ 页面导航  │        页面预览             │  页面属性            │
│ 缩略图    │  标题 · 核心信息 · 大图      │  视觉意图 / 版式质量  │
│ 上一页/   │                             │  ─────────────      │
│ 下一页    │                             │  内容适配（4 种）     │
│          │                             │  AI 编辑（8 种）      │
├──────────┴────────────────────────────┴─────────────────────┤
│  状态与版本 · 整套一致性 · 页面内容修订历史                    │
└─────────────────────────────────────────────────────────────┘
```

顶部 **高级模式** 开关默认关闭。关闭时界面使用中文业务术语（如「页面内容」「页面版式」）；开启后会显示 SlideSpec、LayoutPlan 等内部名称，供开发者或深度用户排查问题。

## 推荐流程

### 1. 选择项目与汇报

在顶部选择 **当前项目** 与 **汇报**。若列表为空，请先到 **项目工作台** 创建项目并生成页面内容。

右侧指标说明：

| 指标 | 含义 |
|------|------|
| 版式就绪 | 已生成页面版式的页数 / 总页数 |
| 预览就绪 | 已有预览图（线框或截图）的页数 / 总页数 |
| 导出状态 | 可导出 / 待生成版式 / 有问题 等 |

### 2. 生成版式

点击 **生成版式**。系统会为每页自动：

- 推断页面视觉意图
- 选择版式类型与变体
- 校验版式质量并尝试修复
- 生成线框预览；若环境就绪，同时导出 PPTX 并生成截图预览

生成过程可在页面底部查看进度。若启用了后台工作流（默认开启），任务会在后台运行，你可以继续浏览其他页面。

### 3. 逐页浏览与检查

在左侧 **页面导航** 中：

- 点击缩略图或页码按钮切换页面
- ✅ 表示版式校验通过；⚠️ 表示有问题需关注
- 缩略图下方标注「版式线框」或「截图预览」

中间 **页面预览** 显示当前页大图、标题与核心信息。版式质量分数与通过/需修复状态显示在右上角。

右侧面板 **版式候选** 可在一键生成的多个方案间切换，无需离开 Studio 打开视觉设计页。

### 4. 调整单页（可选）

根据需要在右侧面板操作：

- **内容适配** — 改文字与结构（见下文）
- **AI 编辑** — 改版式、留白、主图等视觉表现（见下文）

每次修改后会重新校验版式并记录修订，支持撤销上一步。

### 5. 导出 PowerPoint

当 **导出状态** 为「可导出」且 **导出 PPTX** 按钮可用时，点击导出。完成后界面会显示文件路径，文件位于项目的输出目录（默认 `data/outputs/`）。

> 若按钮灰色不可用，说明仍有页面未完成版式生成，或存在阻断性问题。请先生成版式，或在 **页面属性** 中查看「版式问题」。

### 切换候选版式

右侧面板 **版式候选** 区域提供：

| 操作 | 说明 |
|------|------|
| **重新排版** | 为当前页重新生成多个候选版式 |
| **检查问题** | 查看当前页版式校验结果 |
| **选择候选版式** | 在已生成的多个方案间切换 |
| **切换到此版式** | 应用所选候选并更新预览 |

若列表为空，请先点击 **重新排版** 或顶部 **生成版式**。

## 内容适配

**内容适配** 调整当前页的**文字与结构**（页面内容层），修改后系统会尝试重新生成版式并记录修订。

| 按钮 | 效果 |
|------|------|
| 缩短文字 | 压缩正文，保留核心信息 |
| 转为要点 | 将长段落整理为条目 |
| 拆分页面 | 将内容拆成两页（适合信息过载页） |
| 突出核心信息 | 强化标题区核心结论 |

也可在文本框用自然语言描述，例如「缩短这段文字」「拆成两页」，点击 **应用内容适配**。

## AI 编辑

**AI 编辑** 调整当前页的**视觉表现**（版式层），不直接改 SlideSpec 字段。支持 8 种高频操作：

| 按钮 | 效果 |
|------|------|
| 减少文字 | 降低信息密度，突出主信息 |
| 放大主图 | 增大主视觉区域 |
| 增加留白 | 疏朗排版 |
| 切换版式 | 更换版式类型（可说「切换到图纸版式」「换主视觉页」） |
| 设置主图素材 | 指定 hero 素材 |
| 移除素材 | 从版式中移除指定素材 |
| 锁定元素 | 锁定某元素位置，后续重排不再移动 |
| 撤销上一步 | 恢复该页上一版视觉状态 |

自然语言示例：

- 「减少文字」
- 「放大主图，增加留白」
- 「切换到图纸主导版式」
- 「撤销」

每次应用修改后会自动校验版式。若校验未通过，界面会提示具体问题（如文字溢出），可继续调整或撤销。

### 版式类型（切换版式时可指定）

| 用户说法示例 | 版式类型 |
|--------------|----------|
| 主视觉 | 主视觉页 |
| 图纸 / 平面 / 剖面 | 图纸主导 |
| 证据 / 照片网格 | 证据板 |
| 比较 / 对比 | 案例比较 |
| 流程 / 分期 | 过程叙事 |
| 分析图 | 分析图 |
| 指标 / 数据 | 指标看板 |
| 策略 / 卡片 | 策略卡片 |
| 文字 / 论述 | 文字论述 |
| 混合 / 图文 | 混合画布 |

完整技术说明见 [Layout Families](visual/layout-families.md)。

## 状态与版本

页面底部 **状态与版本** 面板汇总：

| 信息 | 说明 |
|------|------|
| 页面数 / 版式就绪 | 整套进度 |
| 导出状态 | 是否可导出 PPTX |
| 整套一致性 | 跨页节奏与风格一致性评分（Deck QA） |
| 当前页视觉修订 | 本页 AI 编辑 / 内容适配产生的修订条数 |
| 页面内容修订历史 | Brief / Storyline / 页面内容的 lineage 历史 |

**撤销** 仅针对视觉编辑链：在 AI 编辑中点击 **撤销上一步**。内容适配的修订可在「页面内容修订历史」中查看 lineage，Studio 内暂不提供逐步 Undo。

## 与项目工作台的关系

```
项目工作台                          汇报工作室
─────────────                      ─────────────
导入资料 · 事实账本                 浏览全部页面
Brief → Storyline → 页面内容  ──→   生成版式 · 预览
审核与编辑内容                      内容适配 · AI 编辑
JSON / Markdown 导出                导出 PowerPoint
```

内容在工作台生成与审核；版式在工作室生成、预览与导出。两边共享同一项目与汇报，切换页面无需重新导入。

## 与视觉设计的关系

| 能力 | 汇报工作室 | 视觉设计 |
|------|:----------:|:--------:|
| 三栏浏览 + 缩略图导航 | ✅ | — |
| 一键生成全套版式 | ✅ | ✅ |
| 单页 NL 视觉编辑（8 意图） | ✅ | 部分（预设按钮） |
| 内容适配（4 动作） | ✅ | — |
| 视觉方向审核门 | — | ✅ |
| 多候选版式列表切换 | ✅ | ✅ |
| 导出 PPTX | ✅ | ✅ |

**日常编辑以汇报工作室为主**；需要审核 ArtDirection 或在 3 个候选版式间精细挑选时，打开 **视觉设计**。详见 [视觉设计用户指南](visual/user-guide.md)。

## 常见问题

**还没有项目或汇报？**  
先到 **项目工作台** 创建项目，运行 Brief → Storyline → 页面内容，再返回工作室。

**左侧缩略图显示「无预览」？**  
点击 **生成版式**。线框预览在版式生成后立即可用；PPTX 截图预览需成功完成导出步骤。

**导出 PPTX 按钮不可用？**  
需全部页面均有版式且通过导出就绪检查。查看顶部「版式就绪」是否等于总页数。

**导出失败或没有文件路径？**  
确认 Node.js 已安装，并在 `archium/infrastructure/renderers/pptxgen` 执行 `npm install`。失败信息会显示在界面上，不会静默丢失。

**版式质量显示「需修复」？**  
展开 **版式候选** → **页面问题** 查看详情。常见项：文字溢出、字号过小、图纸被裁切。可尝试 AI 编辑中的「减少文字」「增加留白」，或在 **版式候选** 中切换其他方案。

**图纸显示占位框？**  
检查 **素材板** 中该页是否已绑定正确图纸素材，路径是否可解析。

**想换版式变体而不是版式类型？**  
在 **版式候选** 面板选择其他候选并点击「切换到此版式」，或点击「重新排版」刷新列表。

**高级模式什么时候开？**  
排查问题、对照测试用例、或需要看到内部 ID 与字段名时再开启。普通汇报生产保持关闭即可。
