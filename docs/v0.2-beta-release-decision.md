# v0.2 Beta 发布决策

> **最后更新：** 2026-07-18  
> **决策：** ⛔ **暂不发布 Beta 标签**（工程门槛部分满足；真实用户闭环未完成）

## 一句话

Beta 不是「工程师觉得可以了」，而是「非开发用户在真实项目上走通主链，且修改成本可量化」。当前 **B10 未完成**，因此 **不能** 基于现有数据打 `v0.2-beta` 标签。

## 何时可以打 Beta 标签

**全部满足**时，将 `pyproject.toml` / README 版本升为 `v0.2-beta.1` 并打 git tag `v0.2-beta.1`：

1. 上表 **剩余 Beta blocker** 全部为 ✅；
2. `summarize_beta_rehearsal.py` 输出中 `participants_non_dev >= 1` 且 `total_edit_minutes > 0`；
3. `beta_blocker` 开放 issue 数为 **0**；
4. `master` 上 CI + Windows nightly 连续绿 ≥3 天（或 RC tag 上全绿）。

## 发版后仍属 Manual 的能力

见 [beta-platform-support-matrix.md](beta-platform-support-matrix.md)：**中文字体像素级、PowerPoint 主观版式** 仍为人工复核，不因 Beta 标签而变为自动化保证。
