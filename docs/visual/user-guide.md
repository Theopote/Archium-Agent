# 视觉设计用户指南

> **入口在制作 → 工作室**（侧栏路径 `edit`；旧 URL `studio` 会跳转到此）。  
> 三栏浏览、NL 视觉编辑、内容适配、版式生成与 PPTX 导出均在工作室完成。详见 [汇报工作室用户指南](../studio-user-guide.md)。

本页说明与工作室相关的 **视觉编排**（ArtDirection → Intent → LayoutPlan）与审核门行为。

## 前置条件

1. 已有项目，并经 **大纲 → 生成**（或设置中的深层「项目工作台」）产出 Brief / Storyline / SlideSpec。
2. （可选）配置 LLM API Key；未配置时使用规则生成，仍可完成编排。
3. 导出原生 PPTX 需已安装 Node.js，并在 `archium/infrastructure/renderers/pptxgen` 运行过 `npm install`。

## 推荐流程

1. 打开侧栏 **制作 → 工作室**，选择项目与汇报。
2. 在工具栏选择场景预设，点击 **生成版式**（后台跑 VisualWorkflow）。
3. 产品默认跳过 ArtDirection 审核门以加快迭代；若工作流在视觉方向门暂停，可在工作室顶部 **活动中心 → 状态** 中批准或「批准并继续工作流」。
4. 在检查器 **布局** 中查看 / 切换候选版式；用画布与 AI 工作区微调页面。
5. 导出请走 **交付** 页（工作室工具栏也有「打开交付页」入口）。

## 审核门

当视觉工作流在 `art_direction` 门暂停时：

- **批准视觉方向**：仅批准，不自动续跑
- **批准并继续工作流**：批准并恢复后续意图 / 版式 / 渲染
- **根据反馈重新生成**：写入反馈后重生 ArtDirection

工作室日常「生成版式」默认不开启该门（一次跑完到 finalize）。

## 单页预设与候选

工作室检查器「布局」支持候选版式与预设重排（减少文字、放大主图、增加留白、图纸优先等）。  
预设会更新 VisualIntent 并重新生成候选 LayoutPlan，**不会**在 UI 里手写坐标。

## 与制作五阶段的关系

- **资料 / 大纲 / 生成**：内容管线（Brief → Storyline → SlideSpec）
- **工作室**：看见与调整（ArtDirection / Intent / LayoutPlan / Scene / 画布）
- **交付**：导出 PPTX/PDF 与质量检查

深层「项目工作台」（`workspace`）仍可通过 **设置 → 开发者与验收** 打开，含完整审核 / 历史工具，不在侧栏。

## 常见问题

**只有 JSON 没有 PPTX？**  
在交付页导出，并确认 PptxGenJS 依赖已安装；失败时警告会出现在结果摘要。

**图纸被裁切？**  
`drawing_focus` 与图纸元素默认 forbid crop；若仍异常，检查资产路径是否解析成功（缺图会显示占位框）。

**想换版式但不想重跑整套？**  
在工作室检查器「布局」选候选或点预设重排即可。
