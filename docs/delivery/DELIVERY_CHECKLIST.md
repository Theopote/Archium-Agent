## 🔧 部署建议

### 1. 最小部署（无 LLM）
```python
# 仅使用增强规则解析器
service = VisualEditService(session, use_llm=False)
```
- ✅ 无额外配置
- ✅ 无额外成本
- ✅ 支持大部分常见指令

### 2. 完整部署（含 LLM）
```python
# 配置 LLM
settings.llm_provider = "openai"
settings.llm_api_key = "sk-..."

# 启用完整功能
service = VisualEditService(session, use_llm=True)
```
- ✅ 支持所有复杂指令
- ⚠️ 需要 API 密钥
- ⚠️ 有使用成本

### 3. 推荐配置
- 开发环境：启用 LLM（便于测试）
- 生产环境：根据用户群和预算决定
